# Текстовая приключенческая игра «Лабиринт сокровищ»

Текстовая приключенческая игра «Лабиринт сокровищ»: игрок блуждает по комнатам, собирает артефакты в инвентарь, разгадывает головоломки и охотится за главным призом, избегая ловушек.

## Установка poetry (если ещё не установлен)

Откройте терминал и выполните:

```
sudo apt install python3-poetry
poetry config virtualenvs.in-project true
```

## Установите зависимости проекта (в корневой дериктории):

откройте терминал и выполните:

```
poetry install
```

## Зауск игры (в корневой дериктории):

Откройте терминал и выполните: 

```
poetry run python -m labyrinth_game.main
```
